# ☩ ЦерковноСлавянский Питон ☩

**Версия:** 3.3 — «Святая Троица»

> «И рече Господь: да будет свет, и бысть свет. И рече Господь: да будет Питон, и бысть Питон. И рече Господь: да будет ЦерковноСлавянский, и бысть ЦерковноСлавянский.»
> — Книга Бытия, глава неизвестная

---

## Что сие есть?

ЦерковноСлавянский Питон (ЦСП) — есь диалект языка Питон, в коем ключевыя словеса и встроенныя функции заменены на церковнославянския, яко да программирование было не токмо полезно, но и духовно.

---

## Как сотворити исполняемый транспилятор?

### Первый путь (благословенный):

```bash
chmod +x csp
./csp program.csp
```

### Второй путь (допустимый):

```bash
python3 csp program.csp
```

## Священныя словеса

| ЦСП | Питон | Смысл |
|-----|-------|-------|
| `аще` | `if` | ежели |
| `иначе` | `else` | аще не так |
| `аще_же` | `elif` | ежели же |
| `по` | `for` | по всем |
| `дондеже` | `while` | дондеже есть |
| `сотвори` | `def` | сотвори функцию |
| `воздаждь` | `return` | возврати |
| `чин` | `class` | чин (класс) |
| `возгласи` | `print` | возгласи во весь голос |
| `внемли` | `input` | внемли (выслушай) |
| `число` | `int` | преобразуй в число |
| `дробь` | `float` | преобразуй в дробь |
| `слово` | `str` | преобразуй в слово |
| `список` | `list` | сотвори список |
| `словник` | `dict` | сотвори словник |
| `мери` | `len` | измери длину |
| `предел` | `range` | предел (диапазон) |
| `покусай` | `try` | покусай (попробуй) |
| `яко_аще` | `except` | ежели не удасться |
| `конечно` | `finally` | конечно (в конце) |
| `воздвигни` | `raise` | воздвигни исключение |
| `внеси` | `import` | внеси (импортируй) |

| `истина` | `True` | истина |

| `ничтоже` | `None` | ничто |
| `и` | `and` | и |
| `или` | `or` | или |
| `не` | `not` | не |
| `в` | `in` | в (внутри) |
| `есть` | `is` | есть (является) |

## Пример программы

```csp
внеси random яко ИскушениеЗмииное

чин СвятойКалькулятор:
    сотвори __сотвори__(сам):
        сам.чудеса = 0

    сотвори молитвенное_сложение(сам, а, б):
        шаг = 0.001
        интеграл = 0
        х = 0
        дондеже х < 1:
            интеграл = интеграл + (а + б) * шаг
            х = х + шаг
        сам.чудеса = сам.чудеса + 1
        воздаждь округли(интеграл, 2)

кальк = СвятойКалькулятор()
результат = кальк.молитвенное_сложение(5, 3)
возгласи(f"По молитвам святых: 5 + 3 = {результат}")
```

## Книга Грехов (обработка ошибок)

При грехе программном транспилятор возглашает осуждение по Книге Грехов:

| Ошибка | Осуждение |
|--------|-----------|
| `NameError` | **АНАФЕМА** — Имя призвано дерзновенно |
| `SyntaxError` | **ЕРЕСЬ** — Богохульное нарушение синтаксиса |
| `IndentationError` | **ЕРЕСЬ ОТСТУПНИЧЕСТВА** — Отступление от праведного пути |
| `ZeroDivisionError` | **БОГОХУЛЬСТВО** — Деление на ничтоже |
| `RecursionError` | **ВЕЧНОЕ ПРОКЛЯТИЕ** — Дух заблудился в бесконечном круге |
| `TypeError` | **СМЕШЕНИЕ** — Дерзкое смешение несмешиваемых |
| `ValueError` | **НЕЧЕСТИЕ** — Значение не принято |
| `FileNotFoundError` | **УТРАТА** — Свиток не обретён |
| `ImportError` | **ИЗГНАНИЕ** — Модуль отверг призыв |
