## Что есть ЦСП?

**ЦерковноСлавянский Питон** (сокращённо **ЦСП**) — сие есть священный диалект языка Питона, в нёмже вся словеса писаны суть по образу церковнославянскому.

Транспилятор, иже приемлет код, написанный словами древними и благочестивыми, и претворяет его в чистый Python, дабы железо исполнило волю твою.

Вместо «if» — `аще`, вместо «else» — `иначе`, вместо «def» — `сотвори`, вместо «print» — `возгласи`, вместо «int» — `число` и тако далее.

Всё сие дабы душа программиста не томилась в мире суетном, но пребывала в молитвенном настроении даже за писанием кода.

---

## Како запустити сие чудо?

1. **Скачай** файл `csp` из сего хранилища.

2. **Сделай его исполняемым** (по желанию):
   ```bash
   chmod +x csp
   ```

3. **Сохрани** свою программу с расширением `.csp` (например, `молитва_калькулятора.csp`).

4. **Запусти** прямо из терминала тако:
   ```bash
   ./csp твоя_программа.csp
   ```
   Или, аще не сделал исполняемым:
   ```bash
   python3 csp твоя_программа.csp
   ```

Транспилятор сам покажет:
- исходный код на ЦСП,
- код, претворённый в Python,
- и результат исполнения (или осуждение греха, аще где прегрешил).

## Доступные словеса (краткий свиток)

### Условия:
| ЦСП | Питон |
|-----|-------|
| `аще` | `if` |
| `иначе` | `else` |
| `аще_же` | `elif` |

### Циклы:
| ЦСП | Питон |
|-----|-------|
| `по` | `for` |
| `дондеже` | `while` |
| `престани` | `break` |
| `продолжи` | `continue` |

### Функции и прочее:
| ЦСП | Питон |
|-----|-------|
| `сотвори` | `def` |
| `воздаждь` | `return` |
| `чин` | `class` |
| `возгласи` | `print` |
| `внемли` | `input` |
| `число` | `int` |
| `дробь` | `float` |
| `истина` / `лжа` / `ничтоже` | `True` / `False` / `None` |

И многое иное: `внеси` (import), `покусай` (try), `воздвигни` (raise) и прочая премудрость.

Полный список смотри в самом файле `csp` в словаре замен.

## Пример: Простой калькулятор на ЦСП

Сохрани сие в файл, скажем, `калькулятор.csp` и запусти:
```bash
./csp калькулятор.csp
```

```csp
# Молитва калькулятора
возгласи("Благослови, душе моя, сие исчисление")

Иже_еси = внемли("Введи первое число: ")
Оно_же = внемли("Введи второе число: ")

# Претвори в числа
первое = число(Иже_еси)
второе = число(Оно_же)

возгласи("Что сотворити хощеши?")
возгласи("1 — сложити")
возгласи("2 — отъяти")
возгласи("3 — умножити")
возгласи("4 — разделити")

Деяние = внемли("Введи число деяния: ")

аще Деяние == "1":
    воздаждь = первое + второе
    возгласи("Сумма их есть: " + слово(воздаждь))
аще_же Деяние == "2":
    воздаждь = первое - второе
    возгласи("Разность их есть: " + слово(воздаждь))
аще_же Деяние == "3":
    воздаждь = первое * второе
    возгласи("Плод умножения: " + слово(воздаждь))
аще_же Деяние == "4":
    аще второе == 0:
        возгласи("О, безумие! Делити на ничтоже — богопротивно!")
    иначе:
        воздаждь = дробь(первое) / дробь(второе)
        возгласи("Часть их есть: " + слово(воздаждь))
иначе:
    возгласи("Не разумею сего деяния. Покайся и введи вновь.")
```

Переменные в примере носят имена в духе церковнославянском: `Иже_еси`, `Оно_же`, `Деяние`, `воздаждь` (результат) — дабы ничто не нарушало благолепного настроения.

## Како творити свои программы?

Пиши код обычными буквами кириллицы, используя слова из словаря ЦСП.

Транспилятор заменит их на питоновские, сохраняя строки, комментарии и f-строки как есть.

Ошибки будут обличены в строгом, но милостивом церковном стиле (с ☩ и анафемой, где подобает).
